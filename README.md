# API for Vigilante Web Heist

## How to run

Prerequisites:

* Golang is installed
* Docker is installed

#### Steps to run:

**With Golang**
1. Run `go build` to compile the executable from the file "main.go"
2. Run the executable named "api" (ex: if using mac run `./api`)

**With Docker**
1. Run `docker run -p 8080:8080 rafibarash/csci4950tgt:api`
